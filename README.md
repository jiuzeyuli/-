width 翻译：宽度；作用：设置盒子的宽度，单位 px 英文 pixel 像素的意思
height 翻译：高度；作用：设置盒子的高度
margin 翻译：边距，也叫外边距；作用：设置盒子和其他盒子之间的距离
border 翻译：边框；作用：设置盒子边框，默认透明
padding 翻译：填充，也叫内边距；作用：设置盒子边框和内容之间的距离
